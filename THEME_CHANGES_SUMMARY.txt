HOW IT WORKS
================================================================================

Theme Application Flow:
┌─────────────────────────────────────────────────────────────────┐
│ User clicks theme toggle (Settings or Ctrl+K)                  │
│                          ↓                                       │
│ themeStore.setTheme('dark' | 'light')                          │
│                          ↓                                       │
│ Theme store updates reactive state                             │
│                          ↓                                       │
│ Theme store applies .dark-mode class to <html> and <body>      │
│                          ↓                                       │
│ CSS variables automatically update                             │
│                          ↓                                       │
│ All components using CSS variables update instantly            │
└─────────────────────────────────────────────────────────────────┘

Synchronization:
- Both toggles use the same useThemeStore()
- Changes in one location instantly reflect in the other
- Theme preference persists across sessions via localStorage
- No manual sync needed

================================================================================
USAGE EXAMPLES
================================================================================

For Users:
1. Go to Settings → Click "Light Mode" or "Dark Mode"
2. Or press Ctrl+K → Search "Toggle Dark Mode" → Select

For Developers:
1. Use CSS variables in components:
   background: var(--color-background);
   color: var(--color-text);

2. Use theme store in TypeScript:
   const themeStore = useThemeStore()
   themeStore.toggleTheme()
   themeStore.setTheme('dark')

3. Check current theme:
   const isDark = themeStore.isDark()

QUICK START FOR DEVELOPERS
================================================================================

To use the theme system in a new component:

1. Import the theme store:
   import { useThemeStore } from '@/stores/theme'

2. Use CSS variables in your styles:
   <style scoped>
   .my-component {
     background: var(--color-background);
     color: var(--color-text);
   }
   </style>

3. That's it! Your component will automatically support both themes.

For more details, see THEME_USAGE_GUIDE.md

================================================================================
SUPPORT & TROUBLESHOOTING
================================================================================

Issue: Theme not applying to new component
Solution: Use CSS variables instead of hardcoded colors

Issue: Text not readable in dark mode
Solution: Check that text uses --color-text or --color-heading

Issue: Theme not persisting
Solution: Check browser localStorage is enabled

Issue: Sync issues between toggles
Solution: Ensure both components use useThemeStore()

For more troubleshooting, see THEME_USAGE_GUIDE.md
